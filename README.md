# 📊 Análise de Retenção de Clientes no E-commerce

Análise de retenção de clientes em e-commerce, identificando que **97% dos clientes realizam apenas uma compra** e destacando oportunidades de crescimento através de estratégias de recompra.

---

## 📌 Sobre o Projeto

Este projeto analisa o comportamento de clientes em um e-commerce com foco em retenção, utilizando **SQL para transformação dos dados** e **Power BI para construção de dashboards orientados à tomada de decisão**.

---

## 🎯 Problema de Negócio

A retenção de clientes é um dos principais desafios em modelos de e-commerce, impactando diretamente o crescimento sustentável.

Neste contexto, o projeto busca responder:

* Os clientes retornam após a primeira compra?
* Qual o nível de retenção da base?
* Clientes recorrentes geram mais valor?

---

## 🧠 Principais Insights

* 📉 **97% dos clientes realizam apenas uma compra**, evidenciando baixa retenção.
* 🔁 Apenas **3,12% dos clientes retornam para novas compras**.
* 💰 Clientes recorrentes apresentam **maior valor médio**, indicando alto potencial de crescimento.
* 📊 O negócio depende fortemente da aquisição de novos clientes.

---

Os resultados indicam uma grande oportunidade de crescimento através de estratégias de retenção, como campanhas de recompra, programas de fidelidade e personalização de ofertas.

---

## 📌 Observação Importante

O dataset utilizado apresenta baixa recorrência de compras, o que inviabiliza uma análise tradicional de churn.

Dessa forma, a abordagem foi adaptada para análise de **retenção e comportamento de recompra**, garantindo maior aderência ao contexto dos dados.
